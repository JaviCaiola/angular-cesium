Proyecto de servidores y ecosistema web de SGE + implementacion de cesium

Presentacion compuesta con la finalidad de aclarar el rumbo que va tomando el proyecto y como funcionaria en produción.

OBJETIVOS GENERALES

  • Pagina web SGE (Beta)
  • Conexion a DNS personalizada
  • API Gestion de usuarios (sin host)
  • Base de datos (MySQL - sin host)
  • Servidor de archivos propio
  • Servidor de host web local

¿Cuál es el plan?

Para el desarrollo de estos objetivos se utiliza un stack herramientas bastante variadas, las cuales se conectan entre si para poder lograr un ecosistema capaz de cumplir con las necesidades del cliente.

Mas especificamente hablamos de tecnologias web basicas como HTML, CSS y Js, con la particularidad de hacer uso de herramientas mas potentes como Angular o Cesium basadas en las anteriores dichas. Tambien en el stack Back-End se utilizan los Frameworks SpringBoot (basado en Java), MySQL para almacenar datos, Microsoft IIS para el host web, Apache Tomcat para el servidor de archivos el cual se encargara de administrar el servidor de recursos.

Y tambien incluir a servicios de terceros como Google Drive, Firebase como almacenamiento en la nube y distribucion web.

Estas son las herramientas que nos sera utiles completar el proyecto, ciertamente cada una tiene su nivel de complejidad pero a medida que se va constriyendo, se van encontrando las soluciones indicadas para cada paso.
